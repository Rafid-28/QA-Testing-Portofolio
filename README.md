# QA Testing Portofolio

Repository ini berisi dokumentasi pengujian software dari beberapa project website yang dikerjakan selama magang dan project tim.

Dokumentasi mencakup:
- Manual Testing
- Blackbox Testing
- Test Case
- Bug Report
- Basic Automation Testing
- UI & Responsive Testing

## Tools & Technologies
- Laravel
- PHP
- MySQL
- HTML
- CSS
- Katalon Studio

## Project

### TV Desa Kota Batu
Dokumentasi testing website berita dan informasi desa berbasis web yang dikembangkan selama program magang di Diskominfo Kota Batu.

### Website Company Profile - Naru Branch Tech
Dokumentasi testing website company profile berbasis web dengan fokus pada pengujian tampilan antarmuka dan responsive layout.
